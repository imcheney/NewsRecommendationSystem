#评价方法
我们定义下面两条规则来衡量一个成功的推荐：

(1)推荐的新闻集如果一半或一半以上在用户点击的新闻集里; (追求推荐的准确率)
(2)用户点击的新闻集如果一半或一半以上在推荐的新闻集里; (追求推荐的召回率)
